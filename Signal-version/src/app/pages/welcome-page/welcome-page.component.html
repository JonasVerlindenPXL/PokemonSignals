A Signal take on Pokemon

In this application you will be able to play a simple version of Pokemon battles.

This project serves as a way to show the impact of the upcoming Signal reactive primitive within the Angular framework. There will be two applications both with the exact same functionalities but different approaches to state management and change detection. I would like to kindly ask every participant to carefully examine the code of each version and give any form of feedback(i'm eager to learn from experienced crafters :D) After playing around with the app itself, you can view the code in THIS repository. Follow the README to continue your adventure!

Thank you for your time!
